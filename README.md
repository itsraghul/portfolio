# Raghul S — Portfolio

Personal portfolio at [raghuls.dev](https://raghuls.dev), built as three explorable worlds:

- 🛰 **Ship's bridge** — sci-fi HUD home, comms console, and a working CRT terminal (try `sudo hire-me`)
- 🏴‍☠️ **The Grand Line** — career voyage told through WANTED posters with rising bounties, and a captain's log
- 🎮 **Classic RPG** — projects as a quest log with a playable boss battle, skills as a character sheet

The nav re-skins per world, pages wipe between them, every sound is synthesized in the browser, and the Konami code works everywhere (↑↑↓↓←→←→BA).

## Stack

Next.js 15 · React 19 · TypeScript · Tailwind CSS · Vitest

## Develop

```bash
npm install
npm run dev    # http://localhost:3000
npm test       # vitest suite
npm run build  # production build
```
